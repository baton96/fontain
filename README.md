# ⛲ fountain ⛲

fountain is a Python tool for font recognition on images

# Installation
```pip install fountain```

# Usage

## API

```python
from fontain import *
```

## CLI
```
fountain [-h] path

Recognize font on image

positional arguments:
  path                  input file path

optional arguments:
  -h, --help            show this help message and exit
```
